Object Oriented Design

With a team of four other students, I implemented Dou Shou Qi, a traditional Chinese board game that is roughly comparable to chess in terms of complexity. Our final delivery allowed users to challenge and play against remote users, with all game activity being stored in a database on a web server. We also built a GUI from scratch and created a tournament functionality.

Algorithms - Honors Project

I worked closely with my instructor and one of my peers to complete an extra project in my algorithms class. The goal of my project was to implement two algorithms to solve the Knapsack problem. The first used a more traditional dynamic programming approach, while the second was built off of the space-efficient approach described in Dr. Sanjay Rajopadhye's recent paper, "Towards a Memory-Efficient Knapsack DP Algorithm".

Tools:
Python
GitHub
